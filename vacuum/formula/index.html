抽气时间与压强的计算

泵的有效抽速

1Se−1Sp=1C\frac{1}{S_e} - \frac{1}{S_p} = \frac{1}{C} Se​1​−Sp​1​=C1​

⇒Se=11Sp+1C=Sp⋅CSp+C=C1+CSp=Sp1+SpC\Rightarrow S_e = \frac{1}{\frac{1}{S_p} + \frac{1}{C}} = \frac{S_p \cdot C}{S_p + C} = \frac{C}{1 + \frac{C}{S_p}} = \frac{S_p}{1+\frac{S_p}{C}} ⇒Se​=Sp​1​+C1​1​=Sp​+CSp​⋅C​=1+Sp​C​C​=1+CSp​​Sp​​

  • SeS_eSe​ - 泵的有效抽速
  • SpS_pSp​ - 泵的抽速
  • CCC - 管道的流导

上式称为真空技术的基本方程。它表明

  1. Se<SpS_e < S_pSe​<Sp​，Se<CS_e < CSe​<C，即泵对容器的有效抽速永远小于泵的抽速或管道的流导；
  2. 若CSp≪1\frac{C}{S_p} \ll 1Sp​C​≪1，则Se≈CS_e \approx CSe​≈C，这就是说管道的流导过分小的话，那么用多大的泵也是无效的；
  3. 若SpC≪1\frac{S_p}{C} \ll 1CSp​​≪1，则Se≈SpS_e \approx S_pSe​≈Sp​，这就是说为了充分发挥泵对容器的抽气作用，希望管道的流导尽可能的大。因此真空管道应该短而粗，千万不能细而长。

真空系统的抽气方程

VdPdt=−SeP+Qd+Ql+Qp+QeV \frac{d P}{dt} = -S_eP + Q_d + Q_l + Q_p + Q_e VdtdP​=−Se​P+Qd​+Ql​+Qp​+Qe​

  • ttt - 抽气时间
  • PPP - 容器中的压强
  • VVV - 容器的体积
  • SeS_eSe​ - 泵对容器的有效抽速
  • dPdt\frac{d P}{dt}dtdP​ - 容器中压强的变化率
  • −VdPdt-V \frac{d P}{dt}−VdtdP​ - 容器内包含的大气，简称为大气；
  • QdQ_dQd​ - 表面解吸（其中有吸附气体和吸收气体）的气体，简称为放气（或释气）；
  • QlQ_lQl​ - 通过微隙漏入的气体，简称为漏气；
  • QpQ_pQp​ - 大气通过容器壁向真空室渗透的气体，简称为渗气（或透气）；
  • QeQ_eQe​ - 液体和固体的蒸气，简称为蒸气。

低真空抽气时间的计算

近似常抽速时抽气时间的计算

不考虑管道影响和漏放气时抽气时间的计算

t=VSlnP0−VSlnP=VSlnP0P=2.3VSlogP0Pt = \frac{V}{S} lnP_0 - \frac{V}{S} lnP = \frac{V}{S} ln\frac{P_0}{P} = 2.3 \frac{V}{S} log\frac{P_0}{P} t=SV​lnP0​−SV​lnP=SV​lnPP0​​=2.3SV​logPP0​​

ttt是P0P_0P0​->PPP的抽气时间。

该公式是计算低真空抽气时间的基本公式。虽然它是在没有管道的条件下导出的,但是对于管道的流导大于泵的抽速的情况下也适用。

不考虑管道影响而考虑漏放气时抽气时间的计算

t=VSPtlnP0−PuP−Put = \frac{V}{S_{Pt}} ln \frac{P_0 - P_u}{P - P_u} t=SPt​V​lnP−Pu​P0​−Pu​​

  • SPtS_{Pt}SPt​ - 泵的理论抽速
  • P0P_0P0​ - 初始压强
  • PPP - 终止压强 (泵入口压强，在不考虑管道的情况下PPP就是容器出口的压强)
  • PuP_uPu​ - 极限真空度，对于某一确定的真空系统，PuP_uPu​是个常值
  • QloQ_{lo}Qlo​ - 漏放气的流量

Pu=QloSPtP_u = \frac{ Q_{lo} }{ S_{Pt} } Pu​=SPt​Qlo​​
